Rules

Discover the hidden code!

The number to be guessed is :

- a 4 digit number

- don't start from 0

- contains all different digits

If you guessed a digit BUT it isn't on the correct place- you'll receive a COW.

If you guessed a digit AND it is on the correct place - you will receive a BULL

4 bulls means you have won the game!

The player gets limit on chances depending on level of difficulty.
